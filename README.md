# Shopping Application MongoDB Database

A MongoDB database for a shopping application.

3 collections:
* Customers
* Sellers
* Products

Design can be found in the file `Design.tldr`.

### Features

* Add a customer.
* Add a seller.
* Add a product.
* Add a product to the inventory of a seller.
* Update customer info.
